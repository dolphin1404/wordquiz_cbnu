## Installation

To compile and run the WordQuiz application, follow these steps:

```bash
git clone https://github.com/YourUsername/WordQuiz.git
cd WordQuiz
gcc -o wordquiz wordquiz.c
./wordquiz
```

## Usage

Run the application:
```bash
./wordquiz
```
